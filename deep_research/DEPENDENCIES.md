# Deep Research System - Dependency List

To ensure all agents work correctly, please install the following packages:

```bash
pip install crewai==0.80.0
pip install llama-index==0.12.0
pip install langchain==0.3.13
pip install streamlit==1.41.1
pip install google-generativeai==0.8.3
pip install google-search-results==2.4.2
pip install newsapi-python==0.2.7
pip install aiohttp==3.11.11
pip install networkx==3.4.2
pip install matplotlib==3.10.0
pip install reportlab==4.2.5
pip install plotly==5.24.1
pip install beautifulsoup4==4.12.3
```

You can install all of them at once with:

```bash
pip install -r requirements.txt
```
